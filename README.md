# Porkpress frontend
This is a frontend layer of whole Porkpress project.

## Used technologies:
* React
* Next.js
* Storybook

## Storybook
One of the process of creating this project was developing all UI components. We used storybook to show how they look like and how they behave.

Run storybook:
```bash
npm run storybook
```
